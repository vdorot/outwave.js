Viewer Class

Defined in: lib/outwave/viewer.js:24
Module: Outwave

Main waveform viewer object

Width and height(x,y) dimensions are used as if for horizontal orientation. If you are using Outwave vertically, exchange the dimensions. Eg. the height option means width on screen.

scrollToTime

(
  • time
  • [positionFraction=0.5]
  • [duration]
)

Defined in lib/outwave/viewer.js:563

Scroll to a certain time

This method should only be used when setCursor is not called periodically(the media player is stopped) or autoscroll is disabled.

Parameters:

  • time Number

    Player position in seconds

  • [positionFraction=0.5] Number optional

    Fraction of the viewport width, to which the time will be scrolled. 0.3 means scrolling the time to 0.3 of viewport width.

  • [duration] Number optional

    Animation duration. If not set, options.autoScrollAnimationDuration will be used;
